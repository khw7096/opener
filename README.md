# Opener
- Simple nk file managing tool for Onset.
- I'm using tkinter and Pmw. because keep light dependency(I think big library pyQT for onset.)

#### Install OSX
```
cd ~ && git clone https://github.com/khw7096/opener.git
cd opener && sh setup.sh
```
